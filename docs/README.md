# GitHub Pages 展示站点

这是为 iOS Figma Demo Builder 项目创建的展示站点。

## 🚀 部署说明

1. 将整个 `docs` 文件夹推送到GitHub仓库
2. 在仓库Settings → Pages中启用GitHub Pages
3. 选择 `main` 分支和 `/docs` 文件夹
4. 站点将在 `https://yourusername.github.io/ios-figma-demo-builder` 可访问

## 📁 文件结构

```
docs/
├── index.html      # 主页面
├── styles.css      # 样式文件
└── package.json    # 配置文件
```

## 🎨 特色功能

- 响应式设计，适配各种设备
- 现代化的UI设计
- 平滑的动画效果
- 清晰的项目信息展示

## 📱 移动端优化

展示页面针对移动设备进行了优化，确保在手机和平板上都有良好的浏览体验。
