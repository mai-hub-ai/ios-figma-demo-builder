# 飞猪酒店搜索页设计规范

## 1. 布局规范

### 1.1 容器边距
- **SearchArea（搜索区域容器）**
  - 左右不留 margin，由父容器控制
  - 内部使用 `p-4` (16px) 内边距
  - 圆角：`rounded-xl` (12px)
  - 背景：白色 + 阴影 `shadow-sm`

### 1.2 行分割线
- 城市选择行、日期选择行、人数选择行之间使用 1px 灰色分割线
- 样式：`border-b border-gray-100`
- 分割线位于行下方，不顶格（由父容器 padding 控制）

### 1.3 子组件规范
- **所有子组件删除外框线**（无 `border`）
- 子组件内部不设置左右 margin/padding，由 SearchArea 统一控制

---

## 2. 字号适配规范（Figma → 实际）

### 2.1 适配原则
- **iPhone 16 Pro 按 2 倍缩放适配**
- Figma 字号 ÷ 2 = 实际 CSS 字号

### 2.2 各组件字号

| 组件 | Figma 字号 | 实际字号 | 说明 |
|------|-----------|---------|------|
| CitySelector 城市名 | 36px | 18px | 最大宽度 120px，超出截断 |
| SearchInput 输入文字 | 32px | 16px | 图标 18×18px 自适应 |
| DatePicker 月份/日期 | 48px/28px | 24px/14px | 统一为 24px |
| GuestPicker 人数文字 | 32px | 16px | 始终显示儿童数 |
| SearchButton 按钮文字 | 30px | 15px | 文案"搜索酒店" |
| HotKeywords 热搜词 | 24px | 12px | - |

### 2.3 字体栈
- **数字**：Fliggy Sans 102
- **中文**：PingFang SC, -apple-system, sans-serif

## 3. 飞猪品牌色系

### 3.1 主色
```css
--color-brand-primary: #FFE033;      /* 飞猪黄 - 主按钮、选中态 */
--color-brand-dark: #F5D600;         /* 深黄 - hover态 */
--color-brand-light: #FFF5B3;        /* 浅黄 - 区间背景、禁用态背景 */
```

### 3.2 使用场景
- **主按钮背景**：`#FFE033`，文字 `#202124`
- **选中态背景**：`#FFE033`，文字 `#202124`
- **区间日期背景**：`#FFF5B3`
- **+按钮边框**：`#FFE033`
- **搜索框 focus ring**：`#FFE033`

### 3.3 禁用色
- 历史日期：`opacity: 0.5`，文字 `#D1D5DB`
- 减号按钮 disabled：`opacity: 0.3`

---

## 4. 浮层规范

### 4.1 高度规范
- **日历浮层**：`h-[60vh]`（固定）
- **城市选择浮层**：`h-[60vh]`（固定）
- **人数选择浮层**：`maxHeight: '70vh'`（自适应，内容撑开）

### 4.2 底部安全距离
- 所有浮层底部按钮区域使用 `pb-safe-bottom`
- 确认按钮与底部保持安全距离，不贴边

### 4.3 交互规范
- 浮层从底部滑出，圆角 `rounded-t-2xl`
- 点击遮罩关闭浮层
- 标题栏右侧有关闭按钮

### 4.4 日历浮层特殊规范
- 支持任意顺序选择两个日期
- 自动解析入住/离店（按时间排序）
- 历史日期（昨天及之前）禁用
- 打开时自动锚定到选中日期
- 选中态无底部标签

---

## 5. 组件尺寸规范

### 5.1 高度规范
| 组件 | 高度 | 说明 |
|------|------|------|
| CitySelector | 50px | - |
| SearchInput | 50px | - |
| DatePicker | 自适应 | 根据内容 |
| GuestPicker | 42px | - |
| SearchButton | 42px | 圆角 66px（胶囊形） |

### 5.2 按钮规范
- **SearchButton**
  - 宽度：100%
  - 高度：42px
  - 圆角：66px（胶囊形）
  - 背景：`#FFE033`
  - 文字：15px，"搜索酒店"

---

## 6. 间距规范

### 6.1 行间距
- 行内元素：`gap-2` (8px)
- 行与行之间：`py-3` (12px) + `border-b`

### 6.2 组件间距
- 热搜词与搜索按钮：`mb-2` (8px)
- 搜索按钮与底部：由父容器控制

---

## 7. 图标规范
- 使用 SVG 图标，不依赖外部库
- 尺寸根据上下文自适应
- 颜色继承父元素或使用 `text-gray-400`
